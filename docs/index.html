A Web Component base

With the ever advancing web platform we can do things that where hard and brittle just a few years ago. Also, browser vendors are committed to becoming more interoperable allowing us to utilise upcoming technologies with confidence that these will land in all future browsers. No more polyfiling and hoping the feature gets supported.

Project "Tivet" is a way to enable these new technologies in solutions where buildtooling stills needs to catch up. Trivet use the following web features to enable this white label library:

Web Components Importmaps Native CSS modules through assertions Constructed stylesheets CSS Layers CSS Custom Properties CSS Subgrid CSS '::part()' CSS Container queries CSS ':has()' CSS Color Mix Toggles and Modals
